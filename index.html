About VisOil

VisOil is a web application used to display real-time drilling data from a wellsite into a detailed interactive line-graph along with features for report creation, printing and adding notes for collaboration. Many exploration and production companies still use static graphs stored locally on desktops. These graphs need to be constantly updated manually by engineers, and have high costs both in time and labor and are prone to human error. With VisOil these issues are elliminated and you are also offered instant access to your drilling-data at anytime and from anywhere over the internet via secured access.

Features

Real-Time

Get up-to-date drilling progress everytime you open the app, and watch in real-time as the data grows. This eliminates the need to constantly download data updates from thrid parties, and drastically increases response time for critical drilling-realated decisions.

Secure

Visoil is backed by Google Firebase authentication and security, ensuring that your data is protected from attackers. All communication betweeen the application and the database are encrypted, and email verification ensures that only the correct users have access to data.

Interactive

Better than traditional spreadsheet-based applications, Visoil allows you to interact with the data: toggle layers on and off, select and zoom to specific areas quickly, and print to pdf without needing to install extra browser plugins

Scalable

Visoil is built to be modular: Is there a feature you want added? Want to customize the look of your graph? All these can be done with minimal effort by our developer team to give you an application that suits your style and business needs
